## 요청 작성 가이드

### 요청 번호 규칙
- `CR-XXX` 형식 사용 (CR = Change Request)
- 순차적으로 번호 부여

### 상태 정의
| 상태 | 설명 |
|------|------|
| 검토 중 | 요청 접수 후 분석 단계 |
| 진행 중 | 개발 진행 중 |
| 대기 | 우선순위 밀림 또는 보류 |
| 완료 | 개발 및 테스트 완료 |

### 우선순위 정의
| 우선순위 | 설명 |
|----------|------|
| 높음 | 핵심 기능 또는 긴급 수정 |
| 보통 | 일반적인 개선 사항 |
| 낮음 | 편의 기능 또는 장기 과제 |
