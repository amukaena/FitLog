## 2. 구현된 화면

### 2.1 캘린더 화면 (메인)
- 월간 캘린더 표시
- 월 이동 (이전/다음)
- 날짜 선택
- 운동 기록 마커 (녹색 점)
- 오늘 날짜 강조
- 선택 날짜 운동 요약 카드

### 2.2 일일 운동 기록 화면
- 운동 제목/메모 입력
- 운동 종목 추가 (바텀시트)
- 이전 기록 복사
- 운동별 세트 편집
- 저장 기능

### 2.3 세트 편집 화면
- 세트 추가/삭제
- 무게/횟수 입력
- 저장 기능

### 2.4 운동 목록 관리 화면
- 카테고리별 필터링 (탭)
- 최근 사용순 정렬
- 커스텀 운동 추가

### 2.5 설정 화면
- JSON 내보내기
- JSON 가져오기
- Google Drive 백업 (UI만)
