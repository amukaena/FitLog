# FitLog - 프로젝트 개요

## 1. 프로젝트 소개

**FitLog**는 캘린더 기반의 운동 기록 관리 앱입니다.
사용자가 매일의 운동을 기록하고, 과거 기록을 쉽게 조회할 수 있습니다.

## 2. 핵심 목표

- 간편한 운동 기록 입력
- 캘린더를 통한 직관적인 기록 조회
- 로컬 저장 + 백업 기능으로 데이터 안전성 확보

## 3. 주요 기능

### 3.1 캘린더 뷰
- 월간 캘린더 표시
- 운동 기록이 있는 날짜 표시 (마커/색상)
- 날짜 선택 시 해당 날짜의 운동 기록 표시

### 3.2 운동 기록
- 운동명 입력 (자주 쓰는 운동 목록에서 선택 가능)
- 세트 수, 횟수, 무게 입력
- 여러 운동을 하나의 날짜에 기록 가능

### 3.3 운동 목록 관리
- 자주 하는 운동 미리 등록
- 운동 카테고리 분류 (가슴, 등, 어깨, 팔, 하체 등)

### 3.4 데이터 백업/복원
- JSON 파일로 내보내기/가져오기
- Google Drive 자동 백업 (App Data 폴더)

### 3.5 홈 화면 위젯
- 4x2 크기 위젯
- 저번주/이번주 (2주간) 운동 여부 표시
- 날짜 클릭 시 해당 날짜 기록 화면으로 이동

## 4. 기술 스택

| 구분 | 기술 |
|------|------|
| 언어 | Kotlin |
| UI | Jetpack Compose |
| 위젯 | Jetpack Glance |
| 아키텍처 | MVVM + Clean Architecture |
| 로컬 DB | Room |
| 의존성 주입 | Hilt |
| 비동기 처리 | Kotlin Coroutines + Flow |
| 클라우드 백업 | Google Drive API (OAuth 2.0) |
| 인증 토큰 관리 | Google Sign-In SDK 자동 관리 (직접 저장 불필요) |

## 5. 타겟 환경

- **용도**: 개인용 (Play Store 배포 없음)
- **최소 SDK**: API 26 (Android 8.0)
- **타겟 SDK**: API 36 (Android 16)
- **화면 방향**: 세로 모드 전용 (Portrait only)
- **지원 기기**: 스마트폰 (태블릿 미지원)
- **테마**: 다크 모드 전용 (녹색 포인트)

## 6. 개발 환경

- **IDE**: Visual Studio Code
- **빌드 도구**: Gradle (CLI)
- **SDK**: Android SDK (Command Line Tools)
- **에뮬레이터**: 실제 기기 사용 (Android 16)
